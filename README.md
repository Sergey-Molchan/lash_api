# Lash Studio — Система онлайн-записи на услуги (ресницы, брови)

Современное веб-приложение для студии красоты с календарём записи, админ-панелью, управлением ценами, галереей, отзывами, финансами и интеграцией с СБП (оплата через QR-код).

## 🚀 Возможности

### 👥 Клиентская часть
- Просмотр услуг, цен, портфолио, контактов на карте.
- Запись на конкретную дату и время с учётом длительности услуги (ресницы 2.5 ч, брови 1 ч, комплекс 3 ч).
- Предоплата 300₽ через СБП (заглушка, легко подключается реальный банк).
- Оставление отзыва с рейтингом (⭐) и эмодзи, проходят модерацию админа.

### 🛠️ Админ-панель
- **Календарь**: визуальное управление рабочими днями и выходными, при установке выходного – предупреждение о существующих записях.
- **Записи**: просмотр, подтверждение, отмена, удаление, просмотр комментариев клиента.
- **Цены**: редактирование стоимости услуг и предоплаты.
- **Контент**: изменение текста на главной странице (заголовок, описание, адрес).
- **Галерея**: загрузка фото с названием и описанием, удаление.
- **Отзывы**: одобрение или удаление отзывов клиентов.
- **Финансы**: статистика по выручке, количеству записей, среднему чеку за месяц.
- **Управление часами**: для каждого дня можно закрыть конкретные часы, установить перерыв, скопировать расписание на неделю.

### 🔒 Безопасность
- Админ-панель защищена сессиями, логин и пароль задаются через переменные окружения (по умолчанию `admin`/`admin123`).
- SQL-инъекции исключены за счёт SQLAlchemy ORM.
- Пароль администратора хранится в хешированном виде.

## 📦 Установка и запуск

### 1. Клонирование репозитория
```bash
git clone https://github.com/Sergey-Molchan/lash_api.git
cd lash-api

2. Настройка окружения

Создайте файл .env на основе .env.example:

bash
cp .env.example .env
Отредактируйте параметры (пользователь БД, пароль и т.д.).
Для локальной разработки PostgreSQL уже запущен в Docker, порт проброшен на 5433.

3. Запуск PostgreSQL и Redis через Docker Compose

bash
docker-compose up -d
4. Установка Python зависимостей (рекомендуется Python 3.11)

bash
python3.11 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
5. Создание таблиц и запуск сервера

Таблицы создаются автоматически при первом запуске.

bash
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
6. Доступ к приложению

Страница	URL
Главная	http://localhost:8000/
Запись клиента	http://localhost:8000/booking
Портфолио	http://localhost:8000/portfolio
Админ-панель (логин)	http://localhost:8000/api/admin/login
Админ-панель (после входа)	http://localhost:8000/api/admin
Учётные данные админа по умолчанию:
Логин: admin
Пароль: admin123 (можно изменить через переменную окружения ADMIN_PASSWORD)

🐳 Запуск через Docker (без локального Python)

bash
docker build -t lash-api .
docker run -d -p 8000:8000 --env-file .env --name lash-api lash-api
